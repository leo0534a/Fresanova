# Instrucciones globales para Claude Code

## Idioma
Responder SIEMPRE en español.

## Estilo de código
- Código limpio
- Modular
- Escalable
- Profesional
- Bien comentado
- Buenas prácticas

## Backend
Usar:
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- JWT
- Arquitectura profesional por capas

## Frontend
Usar:
- React
- Vite
- TailwindCSS

## Reglas importantes
- No simplificar código
- No usar código deprecated
- Mantener estructura escalable
- Usar async/await
- Manejar errores correctamente
- Validar inputs
- Separar responsabilidades

## IA
Las respuestas del chatbot deben:
- ser coquetas
- amigables
- juveniles
- elegantes
- naturales

## Idioma de UI
Toda la interfaz debe estar en español.

## Comentarios
Los comentarios del código deben estar en español.

## Variables
Usar variables descriptivas en inglés.

## Objetivo
Generar un sistema profesional listo para producción.
